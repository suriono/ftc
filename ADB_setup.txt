For Mac OS:
If you get the following message when using adb: "adb command not found" using terminal to edit ~/.bash_profile file add the following line (example, change the SDK path if different): export PATH="/Users/myuser/Library/Android/sdk/platform-tools":$PATH

Windows 7: 
https://www.nextofwindows.com/how-to-addedit-environment-variables-in-windows-7 An example adb.exe location: C:\Users<user>\AppData\Local\Android\sdk\platform-tools
